# SEO, accessibility, and performance

## SEO
Maintain one clear H1, logical H2/H3 hierarchy, canonical URL, Open Graph, social preview data, sitemap, robots, descriptive image alt text, and truthful JSON-LD. Do not keyword-stuff.

## Accessibility
Keyboard access, visible focus, associated labels, sufficient contrast, semantic elements, readable error states, and reduced-motion support are mandatory.

## Performance
Avoid React state updates on every pointer move. Prefer refs, transforms, opacity, and requestAnimationFrame. Optimize images, lazy-load below-the-fold media, reserve dimensions, minimize costly blur layers, and keep mobile effects lighter.
